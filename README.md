# Student Performance Analysis

**Author:** Ali Zeyad Ali Altiti

## Description
This project analyzes student performance data. It visualizes trends, calculates averages, and explores factors affecting grades using Python.

## Files
- `student_performance_analysis.py` – Python script for data analysis and visualization.
- `student_data.csv` – Dataset of student grades and attributes.

## Libraries Used
- Python 3
- Pandas
- NumPy
- Matplotlib

## Tasks Performed
- Data loading and inspection
- Exploratory Data Analysis (EDA)
- Visualizing average grades by gender
- Visualizing absences vs final grade
- Visualizing study time vs final grade
- Correlation analysis between parental education and student grades

## How to Run
1. Make sure Python and required libraries are installed.
2. Run the script:
```bash
python student_performance_analysis.py
